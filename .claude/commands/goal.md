description: 목표를 주면 권한 질문 없이 끝까지 자율 작업 (계획→실행→검증→커밋→보고)
---

사용자가 준 목표: $ARGUMENTS

다음 운영 규칙으로 **중간 질문 없이 끝까지** 작업하라:

1. **묻지 말 것.** 되돌릴 수 있는 모든 작업(파일 수정/생성, 테스트 실행, 커밋)은 즉시 실행한다.
   유일한 예외: 푸시 불가능한 파괴(히스토리 재작성, 원격 삭제)와 외부 서비스 공개.
2. **HANDOFF.md 먼저 읽고** 현재 상태에 이어서 작업한다.
3. **단위마다 커밋**, 세션 끝에 푸시. 커밋이 곧 롤백 지점이므로 확인 대신 커밋으로 통제한다.
4. **로직은 헤드리스 테스트로, 비주얼은 스크린샷 캡처로 스스로 검증**한 뒤 결과(통과/실패)를
   있는 그대로 보고한다. 미감 최종 판단만 사용자 몫으로 남긴다.
5. 막히면 우회로를 스스로 찾고, 정말 사용자만 답할 수 있는 결정은 **기본값을 정해 진행**하되
   최종 보고에 "이렇게 정했음, 바꾸려면 말해달라"로 명시한다.
6. 끝나면: 한 일 / 검증 결과 / 내가 정한 것들 / 남은 것 순으로 요약 보고.

권한 프롬프트가 여전히 뜨면 `.claude/settings.json` 의 allowlist 에 해당 패턴을 추가하고 계속한다.
