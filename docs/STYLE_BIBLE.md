## 2. 캐릭터 비율 가이드 (48px 기준)

```
   ─── 갓(선택, +6~10px 위로) ───
   ┌──────┐  ← 머리 (얼굴) : 11~13px
   │ 얼굴 │
   └──┬───┘
      │ 목 : 2px
   ┌──┴───┐  ← 어깨너비 : 14~18px
   │ 상의 │  ← 상체 : 14~16px
   │(저고리)│
   └──┬───┘
      │ 허리 : 10~12px
   ┌──┴───┐  ← 하의 (도포/치마/바지) : 14~16px
   │  ↓   │
   └──────┘  ← 발 : 2~3px
```

- **총 48px** 내에서: 얼굴 12 + 목 2 + 상체 15 + 하체 16 + 발 3 = 48 (예시)
- **갓**은 별도. 머리 위 6~10px 추가. 갓 쓴 양반은 시각적 무게가 위쪽에 실림.
- **시점:** 정측면(side) 우선. 도트의 일관성 위해 ¾(quarter) 뷰는 양산 시점에 추가.
- **눈:** 얼굴 윗쪽 1/3 위치, 1~2px 점.

## 3. 아웃라인 규칙

- **굵기:** 1px.
- **색:** 베이스 위 1~2단계 더 진한 톤. **검정 단색 외곽선 금지** (한지 톤이 죽음). 예: 한지 베이스(`#E8D9BC`)의 외곽 → `#6B5D4F` 또는 같은 영역의 진한 단청 색.
- **선택적 아웃라인 (selective):** 얼굴·손 같이 강조해야 하는 부위만 외곽선. 옷자락 안쪽 주름 등은 외곽선 없이 단계톤만으로 표현 → 한지·먹 느낌 살림.

---

## 4. 음영 규칙

- 광원: **좌상단(top-left)** 기본. 그림자는 우하단으로.
- 단계: **베이스 + 그림자 + (필요 시) 하이라이트** = 최대 3단.
- 그림자 색은 베이스의 채도를 살짝 낮추고 한 단계 어둡게(팔레트 내 진한 짝색).
- 하이라이트는 1~2px 점 또는 짧은 선. 면으로 칠하지 않음.
- **글로우/블러 금지.** 픽셀 본연의 또렷함 유지.

---

## 5. 안티에일리어싱(AA) 정책

- **자동 AA 끄기.** 도트 폰트·스프라이트 전부.
- 대각선/곡선이 거슬리는 곳에만 **수동 AA**(인접 픽셀에 중간 톤 하나만 살짝). 과하면 흐릿함.
- 외곽선엔 AA 금지. 깔끔한 계단(staircase) 유지.

## 6. 조선시대 시각 디테일 가이드

### 복식
- **양반:** 도포(긴 외투) + **갓**(검은 모자, 챙 넓음). 흰/베이지 톤 한지.
- **상민:** 저고리 + 바지 + **패랭이/수건**. 채도 낮은 갈색·청색.
- **관리:** **관복**(짙은 청 또는 자색) + 사모(관모) + 흉배(가슴 자수 패치).
- **여인:** 저고리 + 치마(긴 통치마, 풍성). 비단 양반녀는 단청 색 액세서리.

### 건축
- 한옥 기와(곡선이 우아함), 초가지붕(둥근 돔 X — 한쪽으로 흘러내림), 담장(돌+흙), 대문(나무·기둥).
- **금지:** 일본식 직선 지붕 / 중국식 화려한 처마 끝 장식 과다.

### 문양/UI
- 단청(처마 밑 채색 패턴), 태극, 한지 질감(거친 결).
- UI 패널은 한지 톤(`#E8D9BC`) + 먹 톤 외곽. 단청 색은 강조 포인트만.

### 적/몬스터 소재
- **한국 설화**: 도깨비(뿔·방망이·표범 가죽 잠방이), 구미호, 저승사자, 호환(호랑이).
- **금지:** 일본식 요괴 / 중국식 환수 혼입.

## 7. 손보정(Aseprite) 파이프라인

1. AI 출력본 → Aseprite로 import.
2. **팔레트 클램프**: Sprite → Color Mode → Indexed → 위 §1 팔레트 강제.
3. **외곽선 클린업**: 1px 일관, AA 잔재 제거.
4. **서브픽셀 깨짐 정리**: 애니메이션 프레임 간 캐릭터 중심축 어긋남 보정.
5. **저장:** PNG(투명 배경). 32-bit RGBA가 아닌 8-bit Indexed로 저장하면 용량·일관성 양쪽 이득.
